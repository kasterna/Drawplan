Drawplan Guide

A browser-based 2D floor plan tool. No installation required — open the app and start drawing immediately.

Overview

Drawplan lets you sketch floor plans directly in the browser. Draw walls, place doors and windows, label rooms with area and floor type, add dimension lines and architect grid lines, then export to PNG or PDF.

All drawings are stored locally as .json files that you save to your own computer. Nothing is sent to a server.

💡 Press Esc at any time to cancel the current action and return to the Select tool.

Interface

Top toolbar

The main toolbar runs along the bottom of the screen. Left side: tools. Right side: display controls.

↖ Select Walls Edit Delete Rooms Opening Measure Grid lines

View controls (secondary toolbar)

Below the tools: Drafting / Polish mode toggle, undo/redo, snap magnet, grid toggle, zoom, unit selector, and theme toggle.

Drafting vs Polish mode

Drafting shows a clean line drawing — ideal for working. Polish adds wall fill and gradient — ideal for presentation.

Room schedule

When rooms are named, a schedule panel appears on the right showing all room names, areas, and a total.

Your first plan

  1. Draw the outer walls — select Walls, click to start, drag to set direction, release to finish. Use Chain mode to draw continuous segments.
  2. Add inner walls — continue drawing. Walls snap to existing corners and edges automatically.
  3. Place doors and windows — select Opening, choose Door or Window, set width in MM, then click on a wall to place.
  4. Label rooms — select Rooms, click inside a closed area, type a name, choose a floor type, and confirm with ✓ or Enter.
  5. Add dimensions — select Measure, click two points, move the mouse to offset the line, click to place.
  6. Export — use the toolbar icons to save as PNG (▤) or vector PDF at 1:1 scale (⬡).

Walls

WallsDraw walls
Click and drag to draw a wall segment. Release to finish. The wall thickness is set in the toolbar (default 100 mm).
ChainChain mode
Click once per segment — no need to hold the mouse. Double-click or press Esc to end the chain.
EditEdit walls
Click a wall to select it. Drag the wall body to move it. Drag the endpoint handles to resize. Type an exact length in the toolbar field.
DeleteDelete
Click any wall, door, window, dimension line, or grid line to delete it.

Angle lock

By default, walls snap to 0° and 90°. Hold Alt while drawing to draw at a free angle.

Exact length input

While drawing, a length field appears at the midpoint of the wall preview. Type a value in the current unit (e.g. 3200 in MM mode) and press Enter to set the wall to that exact length.

💡 Connected walls update automatically when you move or resize a shared endpoint.

Doors & Windows

DoorPlace a door
Hover over a wall to see live placement dimensions. Click to place. The swing arc is shown in the plan.
WindowPlace a window
Same as a door, but shown as a double glass line. No swing arc.

Editing a door or window

With the ↖ Select or Opening tool active, click any door or window to select it. A panel appears with:

  • ⇄ Flip hinge side — moves the hinge from left to right
  • ⇅ Flip swing direction — swings the door inward or outward
  • Width field — type a new width in MM and press Enter
  • × Delete — removes the opening

To move a door or window: click it once to select (yellow outline appears), then click and drag it along the wall.

⚠️ Doors and windows cannot be closer than 100 mm to a wall end or junction. If there is not enough space, a message is shown.

Rooms

Select Rooms and click inside any enclosed area. A name field and floor type selector appear.

  • Type the room name (e.g. Living room)
  • Choose a floor type from the dropdown: Parquet, Laminate, Tiles, Concrete, Vinyl, Carpet, Marble
  • Press Enter or click ✓ to confirm

The room is filled with a colour matching the floor type, and the name, area, and floor type annotation are shown in the plan. The room schedule on the right updates automatically.

Click an existing room label to rename it or change its floor type.

Measure

Select Measure and:

  1. Click the first point — snaps to wall corners, edges, and faces.
  2. Click the second point — the line locks to 90° when near-perpendicular.
  3. Move the mouse to set the offset distance from the wall.
  4. Click to place the dimension line.

Dimension lines use architect convention: diagonal tick marks at each end, text in the middle. Press Esc to cancel a measurement in progress. Use the Delete tool to remove a placed dimension.

💡 Snap markers show which point you are snapping to: a ■ square for a corner, ▲ triangle for an edge, and ● circle for a face.

Grid lines

Select Grid lines, then choose direction in the toolbar:

  • ↕ Vertical — numbered lines (1, 2, 3…)
  • ↔ Horizontal — lettered lines (A, B, C…)

Click anywhere on the canvas to place a line. Labels appear in circles outside the drawing at both ends. Labels are re-sorted automatically by position when a new line is added or an existing one is moved.

Editing grid lines

  • Move — hover over the dashed line (cursor becomes ↔ grab), then drag along the perpendicular axis.
  • Jog (offset label) — in Grid lines mode, drag a bubble sideways. Both bubbles move together by default; hold Alt to move only one.
  • Lock — click the 🔓 icon on a bubble to lock the line. Locked lines cannot be moved or deleted until unlocked.
  • Delete — switch to the Delete tool and click the line or either bubble.

Snap system

The 🧲 magnet button in the toolbar toggles snapping on or off. When active, three snap types are shown by coloured markers:

■
Corner (vertex)
Wall endpoint or outer corner
▲
Edge
Any point along a wall face
●
Face (surface)
Inside the wall thickness

Snapping works on all three lines of a wall: outer face, centre line, and inner face. This lets you measure or draw from wall face to wall face — not just centre to centre.

Hold Alt while drawing to temporarily override the 90° angle lock and draw at a free angle, without disabling snapping.

Tracking lines

While drawing walls, yellow dashed tracking lines appear when the current endpoint is horizontally or vertically aligned with an existing wall corner. A yellow dot marks the reference point.

Units

The unit selector in the toolbar controls how all lengths are displayed and entered:

MMMillimetres — default, most precise for construction
CMCentimetres
MMetres
FT/INImperial — feet and fractional inches (e.g. 9'-6 1/2")

Changing the unit updates all dimension lines, the length input field, and the wall edit field immediately.

Save & export

⭳ SaveSave as JSON
Downloads a .json file with all walls, rooms, openings, dimensions, and grid lines. Share this file with anyone to let them open and continue the drawing.
⭱ OpenOpen JSON
Load a previously saved .json file into the current session.
▤ PNGExport image
Exports the full drawing at 2× resolution. Crops to content — the result is always the same regardless of current zoom or pan position.
⬡ PDFExport vector PDF
Exports a true vector PDF at 1:1 scale — 1 mm in the file equals 1 mm in reality. Ideal for printing or importing into Blender/Bonsai as a reference underlay.
⚠️ PDF export requires an internet connection to load the PDF library (jsPDF + svg2pdf.js).
