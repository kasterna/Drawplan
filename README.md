# Drawplan

A browser-based 2D floor plan drawing tool. No installation, no account, no server — just open and draw.

## Live version
👉 [kasterna.github.io/Drawplan](https://kasterna.github.io/Drawplan)

📖 [Guide — all features explained](https://kasterna.github.io/Drawplan/guide.html)

## Features
- Draw walls with snapping, angle lock and chain mode
- Add doors and windows with live placement dimensions
- Move and resize doors/windows after placement
- Label rooms with area calculation and floor type (parquet, tiles, etc.)
- Dimension lines in architect style
- Grid lines with labels (A/B/C and 1/2/3), lock and jog
- Tracking lines when drawing
- Snap to vertex, edge and face (Solibri-style markers)
- Export to PNG and vector PDF at 1:1 scale
- Save and load drawings as JSON
- Undo/redo (50 steps), dark/light theme, multiple units (mm/cm/m/imperial)

## Usage
Open [the live version](https://kasterna.github.io/Drawplan) in any modern browser — or download `index.html` and open it locally. No dependencies, no build step.

Drawings are saved as `.json` files and can be shared with anyone who has Drawplan open.
